Email Spam Classifier
GitHub Repository
Project Description:

Developed a machine learning model to classify emails as spam or not spam using a labeled dataset with two features: label (0 for not spam, 1 for spam) and text (email content). The email text was preprocessed and converted into numerical features using the TF-IDF Vectorizer. A Multinomial Naive Bayes (MNB) classifier was implemented, achieving an impressive 95% accuracy on the test data.

Technologies Used:
  • Tools: Git, GitHub
  • IDE: Jupyter Notebook
  • Language: Python
  • Libraries: Pandas, Scikit-learn
Features
  • Preprocessing and vectorization of text data for effective machine learning.
  • High-performing spam detection with minimal computational overhead.
  • Clear visualization of classification metrics, including confusion matrix and F1 score.
Outcome

The project demonstrates a strong understanding of text-based data preprocessing and machine learning algorithms, delivering a robust email classification system suitable for deployment.

Audio Classification
GitHub Repository
Project Description:

Developed a deep learning model to classify types of audio using the UrbanSound8k dataset. The audio data was preprocessed by extracting MFCC (Mel-frequency cepstral coefficients) features, which were then fed into a Sequential model built with TensorFlow. The model achieved an 80% accuracy on the test data. For deployment, the trained model was saved as a .keras file and integrated into a Flask-based web application. The web app allows users to upload or record audio for real-time classification.

Technologies Used:
  • Tools: Git, GitHub
  • IDE: Jupyter Notebook, Visual Studio Code
  • Languages: Python, HTML
  • Libraries: Pandas, Scikit-learn, Librosa, TensorFlow, Flask
Features
  • Efficient audio preprocessing using MFCC feature extraction.
  • A robust deep learning model with high accuracy for audio classification.
  • User-friendly web interface enabling audio upload and real-time recording for classification.
Outcome

This project showcases expertise in deep learning, audio signal processing, and web application development, delivering a practical tool for classifying audio types effectively.

Car Price Predictor
GitHub Repository
Project Description:

Created a machine learning model to predict the price of used cars based on key features such as car name, company, year of purchase, price, fuel type, and kilometers driven. The categorical features were encoded using OneHotEncoder, and a Linear Regression model was employed for prediction. To streamline preprocessing and model inference, the pipeline included both transformed columns and the trained model, achieving an 86% accuracy. The trained model was serialized into a .pickle file and integrated into a Flask web application, where users can input car details to get price predictions.

Technologies Used:
  • Tools: Git, GitHub
  • IDE: Jupyter Notebook, Visual Studio Code
  • Languages: Python, HTML
  • Libraries: Pandas, NumPy, Scikit-learn, Flask
Features
  • Effective handling of categorical data using OneHotEncoding.
  • Accurate car price predictions using a streamlined pipeline architecture.
  • A user-friendly web app interface for real-time price estimation based on user inputs.
Outcome

This project demonstrates expertise in machine learning, data preprocessing, and web development, providing a practical tool for estimating the resale value of cars.
